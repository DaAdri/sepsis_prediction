# Prédiction de la Septicémie par des Modèles d'Intelligence Artificielle

Ce projet vise à prédire la septicémie à l'aide de divers modèles d'intelligence artificielle, en utilisant des données médicales temporelles (MIMIC III). Plusieurs approches sont explorées, y compris les réseaux de neurones artificiels, les réseaux de neurones convolutifs, les réseaux de neurones récurrents, et.
